# Glintlock Marketplace

Official plugin marketplace for [Glintlock](https://glintlock.gg) — the AI Game Master for solo TTRPG.

## Install

```bash
claude /plugin marketplace add glintlockGG/glintlock-marketplace
claude /plugin install glintlock
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [glintlock](https://github.com/glintlockGG/glintlock) | Solo TTRPG campaign engine — persistent world state, real dice, oracle tables, voice narration, audiobook generation, and a living world that advances between sessions. |
